Cafe Catastrophy - AJERK

A co-op cooking game. I programmed the interaction system and inventory system
using Scriptable Objects and colliders. It was made in 3 weeks time with a team
of 5 people, including 2 developers.

Link to GitHub repository

I created a combiner script, the counter functionality, the entire inventory
system, the interact ability and the machine timers. I also worked on level-building
and played a big role in the team as SCRUM-Master.

The biggest thing I learned from this project is the interact system. I've built an
inventory with Scriptable Objects before, but never a dynamic interaction system
that would work on different items/machines.
